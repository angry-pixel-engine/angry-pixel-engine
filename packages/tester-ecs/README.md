# Angry Pixel Tester

Angry Pixel testing project.

## How to run

All commands must be executed in the root of the project.

### Build the engine:

```
yarn install
yarn run build
```

or

```
npm install
npm run build
```

### Run the testing game:

```
yarn start
```

or

```
npm start
```

The game will run in http://localhost:8080/
